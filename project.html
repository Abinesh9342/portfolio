Student Management System

OVER VIEW:

The Student Management System is a Python-based application designed to help efficiently manage student information. This system allows users to add, view, update, delete, and search for student details using a structured approach. It is particularly useful for educational institutions or administrators managing student records.

FEATURES:

Add Student Records:

  • Users can input new student details, including ID, name, age, and marks.
  • Validates if the student ID already exists to avoid duplicates.

View Students:

  • Displays all students' details in a structured format.
  • Provides an overview of the stored data for easy access.

Update Students:

  • Allows modifications to existing student details by providing their ID.
  • Users can update the name, age, or marks of a specific student.

Delete Students:

  • Enables deletion of a student record by ID.
  • Provides an option to view all details before deletion to ensure accuracy.

Search Students

  • Users can search for a student by name.
  • Displays the matching student details if found.

USE:

  • Managing student records in schools or colleges.
  • Serving as a foundational project for beginners learning Python.
